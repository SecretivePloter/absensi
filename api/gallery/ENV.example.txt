==================================================================
 ENV VARS untuk Modul Galeri (set di Vercel, BUKAN di-commit)
==================================================================
Vercel Dashboard -> Project "absensi-qr" -> Settings ->
Environment Variables -> tambah satu per satu (Production + Preview):

------------------------------------------------------------------
GDRIVE_CLIENT_EMAIL
  Email service account, contoh:
  galeri-bot@nama-project.iam.gserviceaccount.com

GDRIVE_PRIVATE_KEY
  Private key dari file JSON service account (field "private_key").
  Tempel APA ADANYA termasuk:
    -----BEGIN PRIVATE KEY-----\n....\n-----END PRIVATE KEY-----\n
  (Vercel menerima \n literal; kode otomatis menormalkannya.)

GDRIVE_FOLDER_ID
  ID folder Google Drive tujuan upload.
  Ambil dari URL folder: drive.google.com/drive/folders/<INI_ID_NYA>

SUPABASE_URL
  https://pfvlxlfykdabrwijqqxa.supabase.co
  (untuk verifikasi login user yang upload)

SUPABASE_ANON_KEY
  anon/publishable key Supabase (boleh publik).
------------------------------------------------------------------

Setelah menambah env vars, lakukan redeploy (atau git push) agar
fungsi /api/gallery/* memakainya.

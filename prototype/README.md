## 🚀 Cara Preview

1. Buka folder ini di File Explorer
2. Klik 2x pada file **`IDCardPrototype.html`**
3. Browser akan membuka preview ID card PT ICHIKARA

> **Tidak perlu** `npm run dev`, server, atau koneksi internet.

---

## ⚠️ Catatan Penting

- Folder ini adalah **sandbox terisolasi** — tidak mempengaruhi aplikasi utama sama sekali
- Data yang digunakan adalah **data dummy** untuk keperluan preview desain
- **Jangan hapus** folder ini sebelum fitur disetujui dan di-deploy ke project utama
